# FPS Prototype

Play the Game https://play.unity.com/en/games/ae3a809f-e616-4feb-a0d5-83604e3d2461/fps-prototype

##  About

FPS Prototype is a simple 3D first-person project made in Unity.
The player can move, look around with the mouse, jump, and shoot projectiles.

##  Features

* First-person movement (WASD)
* Mouse look (camera + player rotation)
* Jump system
* Shooting system (projectile-based)
* Rigidbody-based physics

## 🛠 Tech

* Unity (3D)
* C#
* Input System
* Rigidbody

##  Gameplay

The player explores a simple scene using FPS controls.
You can move, look around freely, jump, and shoot objects forward.

The goal of this prototype is to demonstrate core FPS mechanics.
